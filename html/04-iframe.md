# iframe
1. Set width and height
   1. `width`
   2. `height`
2. Dismiss border
   1. `frameborder=0`
3. Load page to iframe by hyperlink
   ```html
   <iframe src="" name="iframe"></iframe>
   <p><a href="https://www.wikipedia.com" target="iframe">Page to load</a></p>
   ```
